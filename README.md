use theme chan, please checkout

``` bash
$ git clone https://github.com/denjones/hexo-theme-chan.git themes/chan
```

modify theme `_config.yml` file and add:

``` yml
duoshuo: sprabbit
jiathis: 1589291
baidu_statistics: d98dab13c098e2a1f6439b963347ad32
```

to serve:

``` bash
$ hexo serve
```

to deploy:

``` bash
$ hexo clean
$ hexo deploy
```
